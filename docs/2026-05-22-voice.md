# 2026-05-22 工作记录

## 语音播放修复 → 实时语音唱歌功能实现

### 1. 编译错误修复
- `List<int>` → `Uint8List.fromList()` 类型转换
- 添加 `import 'dart:typed_data';`

### 2. 音频播放失败排查
- **根因**：录制 `.m4a` (AAC) 但播放 MIME 写死 `audio/mpeg` (MP3)，Android MediaPlayer 解码失败
- **修复**：先尝试 URL 扩展名检测 + `DeviceFileSource`，最终改用 `UrlSource` 直接流式播放（最简单可靠）

### 3. TTS API V1 → V3 迁移
- 端点：`/api/v1/tts` → `/api/v3/tts/unidirectional`（HTTP Chunked）
- 鉴权：`x-api-key` → `X-Api-Key` + `X-Api-Resource-Id: seed-icl-2.0`
- 响应：不再直接写入 bytes，改为解析 JSON chunk → base64 解码 → 拼接音频
- 文件：`lib/services/tts_service.dart`

### 4. 唱歌能力实现（核心）

**新增文件**：`lib/services/realtime_voice_service.dart`
- 豆包端到端实时语音大模型 API（`wss://openspeech.bytedance.com/api/v3/realtime/dialogue`）
- WebSocket 二进制协议编解码（4 字节 header + optional + payload）
- `enable_music: true` + SC2.0 模型 → 歌词自动唱

**架构**：
```
AI 文本 → RealtimeVoiceService (WebSocket)
  → StartConnection → StartSession (character_manifest, enable_music)
  → ChatTTSText (全量文本)
  → TTSResponse (PCM 音频帧) → WAV 编码 → 上传 Supabase → UrlSource 播放
```

**降级策略**：
- 有 Realtime 凭证 → 实时语音（能唱歌）
- 无 Realtime 凭证 → 传统 V3 TTS（不唱歌）
