# 潮汐动态记录按钮 — 设计规格

## 概述

将记录页（`MoodListPage`）底部的"记录心情"FAB 按钮替换为页面中央的潮汐动态按钮，视觉上以心形被海水淹没的程度反映用户最近的心情状态。

## 视觉效果

### 按钮外观

- **形状**: 大圆角方形（120×120px，圆角 28px），位于页面内容区中央
- **心形**: Q 版丰满心形，居中放置。水上部分保持红色（#E53935），水下部分转为海水渐变色
- **海水**: SVG 波浪形状，从底部填充，水位高度由最近记录的能量值驱动
- **背景**: 圆角方形背景使用梦幻渐变色，由最近记录的愉悦度驱动
- **呼吸动画**: 3 秒周期，按钮缩放 1.0→1.04，同时光圈脉冲（2 层半透明环扩大缩小）

### 状态映射

| 状态 | 水位 | 背景渐变 |
|------|------|----------|
| 高能量 + 高愉悦 | ~85% | 暖粉→紫（#FFE0E8→#D5E5FF） |
| 高能量 + 低愉悦 | ~80% | 紫红→蓝（#E8D0E0→#C0D0F0） |
| 低能量 + 高愉悦 | ~25% | 薄荷绿→紫（#D5F5E8→#E8D8F0） |
| 低能量 + 低愉悦 | ~15% | 深海蓝→靛（#C8D8E8→#D0C0E8） |
| 无记录（默认） | ~50% | 紫蓝梦幻渐变（#E8D5F5→#E0E0F5） |

### 映射公式

- **水位** = `((energy + 1) / 2) * 0.8 + 0.1`（energy ∈ [-1, 1] → 水位 ∈ [10%, 90%]）
- **背景色** = pleasantness 在暖色端和冷色端之间插值（pleasantness ∈ [-1, 1]）

## 交互行为

1. **静止态**: 呼吸动画持续运行，波浪缓慢起伏（4-6 秒周期）
2. **按压时**: 波浪频率加快、振幅增大（"涨潮"反馈）
3. **点击后**: 波浪迅速退去（"退潮"），弹出 `LogEditorDialog` 记录对话框
4. **有记录时**: 按钮缩小至 100×100px，圆角 24px，位于记录列表上方居中

## 技术实现

### 文件修改

- `lib/pages/mood_list_page.dart` — 替换底部 FAB 为中央潮汐按钮

### 新增 Widget

- `lib/widgets/tidal_mood_button.dart` — 潮汐动态按钮组件
  - 参数: `energy` (double?), `pleasantness` (double?), `onPressed` (VoidCallback)
  - 使用 `AnimationController` 驱动呼吸动画
  - 使用 `CustomPaint` 或 SVG 绘制波浪和心形

### 数据获取

- 从 `_box` 中获取最近一条记录的 `energy` 和 `pleasantness`
- 若最近记录无这些字段（旧版本数据）或 box 为空，使用默认值

### 布局调整

- 空状态: 今日概览卡片 → 潮汐按钮（垂直居中，占据主要视觉空间）
- 有记录: 今日概览卡片 → 记录列表 → 潮汐按钮（列表下方居中）→ 底部间距

## 设计约束

- 保持现有 `LogEditorDialog` 和分享流程不变
- 不影响批量选择模式
- 保持现有 Material Design 主题兼容
- 心形红色不随愉悦度变化（始终 #E53935）
